## 船舶跟踪定时任务问题

### 问题：agent-browser 访问船讯网失败
- **现象**：搜索框输入 "CSCL GLOBE" 后，下拉搜索结果不显示
- **原因分析**：
  1. 船讯网的搜索功能依赖 JavaScript 动态加载
  2. headless 模式下可能存在兼容性问题
  3. 搜索 API 被网站拦截（返回 Object moved）
- **尝试方案**：
  - `agent-browser fill @e4 "CSCL GLOBE"` - 输入成功但无下拉
  - `agent-browser press Enter` - 无搜索结果加载
  - `curl "https://www.shipxy.com/Ajax/SearchShip.ashx?keyword=..."` - API 拦截
- **临时方案**：使用其他船舶追踪网站（MarineTraffic, VesselFinder 等）

### 已修复
- 将定时任务 timeoutSeconds 从 120 增加到 300 秒
- 优化了任务执行步骤，增加了 wait 和 close 清理
