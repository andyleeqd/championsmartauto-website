## lossless-claw 安装

### 背景
需要安装 Martian Engineering 的 lossless-claw 插件，替代 OpenClaw 自带的滑动窗口压缩机制，实现无损上下文管理（LCM）。

### 问题
1. 之前安装过 0.4.0 版本，但扩展目录被删除
2. 配置文件中有残留配置（plugins.allow 和 plugins.entries 中有 lossless-claw 条目）
3. openclaw plugins install 失败，报错：plugin not found: lossless-claw

### 解决方案
1. 移除配置文件中的 lossless-claw 残留配置：
   - 从 plugins.allow 数组中移除 "lossless-claw"
   - 移除 plugins.slots.contextEngine 配置
   - 移除 plugins.entries.lossless-claw 配置
   - 移除 plugins.installs.lossless-claw 安装记录
2. 重新执行安装
3. 安装后自动配置 contextEngine 插件槽位

### 最终结果
✅ 安装成功！
