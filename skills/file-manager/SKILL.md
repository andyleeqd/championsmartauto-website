# File Manager - 文件管理器

全面的文件和目录管理工具。

## 功能

1. **浏览目录** - 列出目录内容、查看文件信息
2. **文件操作** - 移动、复制、重命名、删除文件和目录
3. **搜索文件** - 按名称、类型、大小、日期搜索
4. **批量操作** - 批量重命名、移动、删除
5. **权限管理** - 查看和修改文件权限
6. **创建文件/目录** - 新建文件和文件夹

## 输出格式

### 目录列表
```

```

### 文件信息
```

大小: 1.2KB
类型: 文本文件
权限: rw-r--r--
修改时间: 2026-03-15 10:30:00
```

### 操作结果
```
✅ 移动成功: A → B
❌ 删除失败: 权限不足
⚠️ 警告: 文件已存在
```

## 注意事项

- ⚠️ 删除操作不可逆，执行前会提示确认
- ⚠️ 系统文件目录需要管理员权限
- ⚠️ 批量操作前会显示预览
- 🔒 敏感路径需要额外确认
